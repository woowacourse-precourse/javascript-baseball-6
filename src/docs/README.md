## 구현할 기능 목록

1. 1에서 9까지 서로 다른 3자리 수를 추출한다.
2. 서로 다른 3자리 수를 입력받는다.
   - 사용자가 잘못된 값을 입력한 경우 종료
3. 입력한 숫자에 대한 결과를 출력한다.
   - 같은 수가 같은 자리에 있으면 '스트라이크'
   - 다른 자리에 있으면 '볼'
   - 같은 수가 전혀 없으면 '낫싱'
   - 모두 맞히면 게임 종료
4. 종료 후 게임을 다시 시작하거나 완전히 종료할 수 있다.
   - '게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.'
   - 1 - 다시 서로 다른 3자리 수 추출
   - 2 - 프로그램 종료
