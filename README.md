기능 구현 목록

1. 컴퓨터에서 랜덤 숫자 생성
2. 사용자에게 숫자 입력 받기
3. 숫자가 3자리 숫자인지 확인
  (1) 3자리 숫자가 아닐 경우 - ERROR 발생
  (2) 3자리 숫자가 맞을 경우
4. 3자리 숫자가 맞을 경우
  (1) 랜덤 숫자와 사용자 숫자 3자리가 모두 맞을 경우 = 3스트라이크
  (2) 스트라이크 또는 볼이 1개 이상일 경우 = ${ball}볼 ${strike}스트라이크
  (3) 모두 맞지 않을 경우 - 낫싱
5. 숫자를 모두 맞췄을 경우 (3스트라이크)
  (1) 게임 다시 시작
  (2) 게임 종료
