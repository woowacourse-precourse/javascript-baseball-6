# 구현할 기능 목록

1. 입력

- 게임 중 입력받은 숫자
- [ ] 숫자
- [ ] 3자리
- [ ] 서로 다른 수

- 게임 종료시 재시작 종료 구분
- [ ] 숫자
- [ ] 1 또는 2

2. 출력

- 메시지 상수화
- [ ] 볼 개수
- [ ] 스트라이크 개수
- [ ] 낫싱
- [ ] 모두 맞춘 경우에 대한 메시지
- [ ] 게임 시작 메시지
- [ ] 게임종료/재시작 메시지
- [ ] 오류 메시지

3. 기능

- [ ] 컴퓨터 3개의 숫자 배열 생성
- [ ] 볼 개수 세는 함수 생성
- [ ] 스트라이크 개수 세는 함수 생성
