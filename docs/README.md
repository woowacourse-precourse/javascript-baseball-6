# 미션 - 숫자 야구

## 기능 구현 사항

- 랜덤 숫자 생성
  - 1부터 9까지의 서로 다른 세 숫자를 무작위로 선택합니다.
- 사용자 입력 처리
  - 콘솔에서 사용자로부터 세 자리 숫자를 입력받습니다.
  - 입력 형식이 올바르지 않으면 오류 메시지를 출력합니다.
- 게임 결과 계산
  - 사용자 입력과 컴퓨터가 선택한 숫자를 비교하여 결과를 계산합니다.
  - 스트라이크와 볼의 개수를 계산합니다.
- 게임 재시작 여부 확인
  - 게임 종료 후 사용자에게 게임을 재시작할지 종료할지 물어봅니다.
  - 입력값에 따라 게임을 재시작하거나 종료합니다.
