## 기능 구현 목록 (ver. 2)

- 1. [ ] 게임 시작 메세지를 출력한다.
- 2. [ ] 컴퓨터가 숫자를 생각한다.
  + 3자리 숫자를 고른다.
  + 서로 다른 숫자를 고른다.
  + 숫자의 범위는 1 이상 9 이하이다.
- 3. [ ] 사용자가 숫자를 입력한다.
  + 3자리 숫자를 고른다.
  + 서로 다른 숫자를 고른다.
  + 숫자의 범위는 1 이상 9 이하이다.
- 4. [ ] 사용자의 숫자와 컴퓨터의 숫자를 비교한다.
  + 같은 수가 같은 자리에 있으면 스트라이크이다.
  + 다른 자리에 있으면 볼이다.
  + 같은 수가 전혀 없으면 낫싱이다.
- 5. 비교 결과를 출력한다.
  + 사용자가 컴퓨터의 숫자를 맞추지 못하면 다시 숫자를 입력한다.
  + 사용자가 컴퓨터의 숫자를 모두 맞추면 게임을 종료한다.
- 6. 재시작 옵션을 선택한다.
  + 1을 선택하면 게임을 다시 시작한다.
    + 컴퓨터가 숫자를 생각하는 단계부터 시작한다.
  + 2를 선택하면 프로그램이 완전히 종료된다.
