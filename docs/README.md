# 구현 기능 목록

- 컴퓨터가 1부터 9 사이 서로 다른 임의의 숫자 3개 선택
- 사용자로부터 숫자를 입력받음
  - 입력이 3자리의 수로 이뤄져있는가?
  - 입력이 숫자로만 구성되어있는가?
  - 유효하지 않은 입력의 경우 `throw` 문으로 예외 발생 및 어플리케이션 종료
- 사용자 입력과 컴퓨터가 고른 숫자를 비교하여 결과 출력
  - 숫자만 동일하면 볼, 숫자와 숫자의 위치까지 동일하면 스트라이크
  - 일치하는 숫자가 하나도 없다면 낫싱
  - 볼과 스트라이크의 개수를 순서대로 표시 ex) 1볼 1스트라이크
  - 3스트라이크면 게임 종료
  - 3스트라이크가 아니라면 다시 입력을 받아 결과 출력
- 게임 종료 후 사용자 입력을 받아 게임 재시작 / 종료 결정
  - 사용자 입력이 1이면 재시작
  - 사용자 입력이 2면 종료
