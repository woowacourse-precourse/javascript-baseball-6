
## 구현할 기능 목록

- 랜덤 수 생성 함수
  - 1~9사이 서로 다른 수 생성
- 사용자 입력 함수
  - 사용자 입력
- 사용자 입력 검사 함수
  - `예외` : 길이 초과
  - `예외` : 숫자가 아닐때
  - `예외` : 중복된 숫자
  - `예외` : 1~9 사이의 숫자가 아닐때
- 숫자 판별 함수
  - 스트라이크 개수, 볼 개수 체크
- 결과 출력 함수
  - 같은 수 없으면 : 낫싱 출력 
  - else : 스트라이크, 볼 개수를 출력
- 게임 종료 여부 함수
  - 스트라이크 3 : 게임 종료
  - else : 사용자 입력 (반복)
- 게임 재시작 함수
  - 사용자 입력
  - 1 : 재시작
  - 2 : 게임 종료
  - `예외` : 입력이 1또는 2가 아닐때
