# 기능 요구 사항

1. 게임 시작
2. 컴퓨터 랜덤 숫자 생성
3. 유저 숫자 입력
4. 컴퓨터와 유저의 숫자 판별후 text 출력
    - 자릿수와 숫자가 모두 일치 시 : 스트라이크
    - 숫자 일치 시 : 볼
    - 일치 없음 : 낫싱
5. 3개의 숫자를 모두 맞힐 시 게임종료
6. 게임 재시작 혹은 종료

# 함수

- 게임 시작 함수
- 출력 관리 함수(게임시작)
- 컴퓨터 랜덤 숫자 생성함수
- 출력 관리 함수(숫자입력)
- 입력 관리 함수(숫자입력)
- 볼 스트라이크 판별 함수
- 결과 문자 반환 함수
- 출력 관리 함수(결과 문자 반환)
- 출력 관리 함수(게임여부)
- 입력 관리 함수(게임여부)
- 출력 관리 함수(게임종료)
