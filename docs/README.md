# 미션 - 숫자 야구

## 🚀 구현할 기능 목록 README.md

###  1. 입출력 기능

#### 입력
- [ ] 서로 다른 3자리 수 입력 받기
- [ ] 게임이 끝난 후 재시작/종료 구분(1과 2 중 하나)

#### 출력

입력한 수에 대한 결과 파트
- [ ] 볼, 스트라이크 개수 표시
- [ ] 하나도 없는 경우
- [ ] 3개의 숫자를 모두 맞힐 경우

게임 진행에 관련된 파트
- [ ] 게임 시작 문구 출력
- [ ] 게임 종료 문구 출력
- [ ] 게임 재시작/종료 선택 문구 출력

###  2. 게임 플레이 내부 로직
- [ ] 랜덤 숫자 생성하기(정답)
- [ ] 생성한 숫자와 입력한 숫자 비교하기 - 판정하기
- [ ] 게임 재시작/종료 구분하기
- [ ] ERROR케이스 처리하기
