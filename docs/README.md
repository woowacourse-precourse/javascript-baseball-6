# 구현할 기능 목록

- [ ] 랜덤 서로 다른 숫자 3개 받아오기
- [ ] 사용자의 숫지 입력받기 (에러처리)
   - [ ] 숫자가 아니거나, 숫자 3개가 아니거나, 중복된 숫자면 에러메세지
- [ ] 랜덤 숫자와 사용자의 숫자 비교해서 메세지 반환
   - [ ] 같은 수가 다른 자리에 있으면 볼처리, 같은수가 같은 자리에 있으면 스트라이크 처리
   - [ ] 다 맞춘 경우 3스트라이크 → 게임 종료
   - [ ] 다 틀릴 경우 낫띵  
- [ ] 게임 재시작/종료 구분하는 함수.
   - [ ] 1: 게임 새로 시작
   - [ ] 2: 게임 종료
