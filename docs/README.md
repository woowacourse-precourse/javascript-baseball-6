## 구현할 기능 목록

* * * 

1. 게임 시작
    * **`Console.print`** 사용

2. 랜덤으로 숫자 생성하기
    * 1에서 9 사이에서 중복되지 않게
      * **`Random.pickNumberInRange()`** 사용

3. 사용자로부터 값을 입력받기
    * **`Console.readLineAsync`** 사용

4. 값이 유효한지 확인
    * 아래의 경우 예외를 발생 (예외 세개를 각 함수로 나눠서 처리)
      * 입력받은 값이 3개가 아닌경우
      * 숫자가 아닌경우
      * 숫자가 중복되는 경우
    * 유효하지 않으면 에러문구 출력
      * **`Console.print`** 사용
      * **`[ERROR] 올바른 숫자를 입력하세요`** 출력

5. 구 판정
    * 같은수가 같은 자리에 있으면 **스트라이크**
    * 다른 자리에 있으면 **볼**
    * 같은 수가 없으면 **낫싱**

6. 판정 결과 출력
    * 각 구의 판정을 합해서 출력
      * **`Console.print`** 사용

7. 정답 판별
    * 3스트라이크인 경우 게임종료
    * 외의 경우 2번으로 되돌아감
    
8. 게임 지속 여부 판단
    * 게임이 종료된 경우 사용자에게 숫자를 입력 받음  
      * **`Console.readLineAsync`** 사용
    * 게임을 새로 시작하려면 1, 종료하려면 2를 입력
      * 1, 2가 아닐 경우 **`[ERROR] 숫자가 잘못된 형식입니다.`** 출력
