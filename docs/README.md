## 📝구현 기능 목록

1. 프로그램 시작

- 게임 시작 문구를 출력하는가
- 1~9 까지의 숫자 3개를 랜덤선택하는가
- 3자리 모두 중복되지 않은 숫자인가

2. 프로그램 진행

- 게임 도중, 1~9까지의 숫자 3자리만을 입력받는가
- 이 외에 잘못된 입력 시, throw문 예외 발생 후 게임 종료 하는가
- 숫자와 자리까지 맞으면 스트라이크, 숫자만 맞고 자리가 다르면 볼, 다 아니면 낫싱

3. 프로그램 종료

- 3스트라이크 시, 게임 종료가 되는가
- 게임 종료 후, 시작(1) or 종료(2) 선택을 할 수 있는가
