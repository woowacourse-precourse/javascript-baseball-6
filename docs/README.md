## 1부터 9까지 서로 다른 수로 이루어진 3자리의 수를 맞추는 프로그램

### 📖구현할 기능 목록

#### 1. 중복되지 않는 랜덤한 숫자 (3자릿수) 만들기

#### 2. 입력한 숫자의 유효성 검사

```
2.1 숫자가 입력이 되었는가?

2.2 3자리수가 입력이 되었는가?

2.3 모두 다른 숫자가 입력이 되었는가?
```

#### 3. 입력한 숫자와의 랜던한 값의 일치 검사

```
3.1 모든 숫자가 일치하는지 확인하기

3.2 자리와 값이 일치하는 수 골라내기 = strike

3.3 자리는 다르지만 값이 일치하는 수 골라내기 = ball
```
