# 📃구현할 기능 목록

- 서로 다른 임의의 수를 생성할 수 있는 "랜덤 숫자 출력 기능"
- 맞혀야 할 세자리 수(정답) 생성
- 각 자리의 숫자가 서로 달라야 함
- 재시작(1) 또는 중단(2) 여부 체크
- 잘못된 입력값 예외 처리
- 숫자가 아닌 값이 포함된 경우
- 세자리 수가 아닌 경우
- 입력값 중 같은 수가 있는 경우
- 핵심 기능: 볼/스트라이크/낫싱 판단
