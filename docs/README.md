# 구현할 기능 목록

1. 서로 다른 수로 이루어진 3자리 숫자를 랜덤으로 뽑는다.

- 이때 뽑을 수 있는 숫자의 범위는 1~9

2. 게임 시작 문구를 출력해야 한다.

- '숫자 야구 게임을 시작합니다.'

3. '숫자를 입력해주세요: ' 문구와 함께 사용자로부터 숫자 3개를 입력 받아야 한다.

4. 잘못된 입력을 받는 경우 에러를 던지고 종료되어야 한다. 이때, 에러메세지는 '[ERROR]' 로 시작해야 한다.

- 아무것도 입력하지 않았을 경우
- 숫자가 아닌 문자,특수문자,공백 등을 입력했을 경우
- 숫자와 함께 문자, 특수문자, 공백 등을 입력했을 경우
- 숫자를 입력하였으나, 3자리보다 많거나 적게 입력했을 경우
- 중복되는 숫자를 입력했을 경우

5. 입력받은 숫자를 검사하여 결과를 출력해야 한다.

- 입력받은 숫자를 1개씩 쪼갠다.
- 1번에서 뽑은 숫자(정답)와 같은 자리에 같은 숫자가 있는지 검사하고, 갯수를 새로운 변수에 담아둔다.
- 자리는 다르지만 1번에서 뽑은 숫자(정답)가 포함되어 있는지 검사하고, 갯수를 새로운 변수에 담아둔다.
- []볼 []스트라이크인지 출력해준다.

6. 3스트라이크가 나올때까지 3~5번 과정을 반복한다.

7. 3스트라이크가 나오면 게임 종료 문구를 출력해야 한다.

- '3개의 숫자를 모두 맞히셨습니다! 게임 종료'

8. 게임 종료 문구가 출력되면 재시작 또는 종료 문구를 출력하고, 사용자로부터 1 or 2를 입력을 받는다.

- '게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.'

9. 1을 입력받으면 다시 게임을 시작한다.

10. 2를 입력받으면 게임을 종료한다.

11. 1, 2가 아닌 입력을 받을 경우 에러를 던지고 종료되어야 한다. 이때, 에러메세지는 '[ERROR]' 로 시작해야 한다.
