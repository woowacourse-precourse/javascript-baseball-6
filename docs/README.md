## 기능 개발 목록

- App Class
  **_Game을 생성하여 사용자가 원할 때 까지 게임을 진행_**

  사용자가 2를 입력할 때 까지 게임을 진행한다.

- Game Class
  **_게임 한 사이클 자체_**

  - 게임 시작 관련
    게임 한 사이클을 시작한다.
    이 때, 게임을 성공하면 재 실행 의사를 묻는다. - !입력값에 대한 예외처리
  - 입력 관련
    사용자에게 1-9의 중복되지 않는 3자리 숫자를 입력받는다. - !입력값에 대한 예외처리
  - 출력 관련
    전달받은 힌트에 대해 출력한다
    1. 정답 - 3스트라이크, 게임 종료 메시지
    2. n볼 m스트라이크

- Computer Class
  **_상대방(컴퓨터)_**

  - 랜덤한 정답 생성
    랜덤하게 1-9의 중복되지 않는 3자리 숫자를 생성한다.
  - 입력값에 대한 result
    입력값에 대한 힌트를 도출한다.
