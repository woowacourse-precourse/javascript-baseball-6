# 구현할 기능 목록

## 1. 게임 시작

- [x] "숫자 야구 게임을 시작합니다."를 출력한다.
- [x] 랜덤한 세자리 수를 뽑아 저장한다 (111~999)

## 2. 유저 숫자 입력

- [x] "숫자를 입력해주세요 : "를 출력한다.
- [x] 숫자를 입력받는다
- [x] 숫자가 입력 조건에 맞는지 검증한다 (1~9의 서로 다른 수로 이루어진 세 자리 숫자)

## 3. 숫자 비교 후 결과 출력

- [x] "n볼 n스트라이크" 형태로 출력한다
- [x] 모두 맞으면 게임을 종료한다. "3개의 숫자를 모두 맞히셨습니다! 게임 종료"

## 4. 게임 종료

- [x] 재시작 여부를 묻는다
- [x] 1을 누르면 재시작, 2를 누르면 완전히 종료한다
