# 기능 구현 목록

### 입력
- [x] 1 ~ 9 3자리 입력
  - [x] 1 ~ 9 숫자 검증
  - [x] 3자리 숫자 검증
  - [x] 중복 검증
- [x] 재시작(1) / 종료(2) 
  - [x] 1 , 2 숫자 검증

### 출력
- [x] 게임 시작 문구 출력
- [x] 숫자 입력 문구 출력
- [x] 힌트 출력 
- [x] 재시작 문구 출력
- [x] 입력값에 대한 에러처리 ("[ERROR]"로 시작)

### 게임
- [x] 서로 다른 숫자 랜덤값 생성
- [x] 입력값, 랜덤값 비교
  - [x] 같은 숫자, 같은 자리이면 스트라이크
  - [x] 같은 숫자, 다른 자리이면 볼
  - [x] 모두 같으면 3스트라이크
  - [x] 모두 아니면 낫싱
- [x] 3스트라이크가 되면 게임 종료
  - [x] 재시작(1) / 종료(2) 
- [x] 에러 발생 시 게임 종료
