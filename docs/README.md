# 구현할 기능 목록 정리

## RandomNumberGenerator

- [x] 3자리 랜덤 숫자를 생성한다

## GameEngine

- [x] 입력받은 값을 배열로 변경한다
- [x] 배열로 변환한 값을 숫자로 변경한다
- [x] 플레이어 숫자가 유효한지 판별한다
- [x] 게임 점수를 반환한다
- [x] 게임 점수를 출력한다

### 예외 처리 사항

- [x] 입력값의 길이가 3이 아니면 에러
- [x] 입력값이 숫자가 아니면 에러
- [x] 입력값이 1~9 사이가 아니면 에러
- [x] 입력값에 중복이 있으면 에러

## GamePlayer

- [x] 게임을 시작한다
- [x] 숫자를 입력한다
- [x] 게임이 종료되면 다시 시작할지, 종료할지 선택한다

### 예외 처리 사항

- [x] 입력값이 '1', '2'가 아니면 에러
