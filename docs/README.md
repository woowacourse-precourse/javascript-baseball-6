## 구현할 기능 목록

### 1. 게임 시작

- 게임 시작 문구 출력 : `숫자 야구 게임을 시작합니다.`
- 컴퓨터의 랜덤값 생성
  - 1에서 9까지 서로 다른 임의의 수 3개

### 2. 게임 진행

- 입력값 요구 문구 출력 : `숫자를 입력해주세요 :`
- 사용자의 입력값 받기
  - 서로 다른 3개의 숫자
  - 예외 처리
- 컴퓨터의 랜덤값과 사용자의 입력값 비교
  - 같은 수가 같은 자리에 있으면 스트라이크
  - 같은 수가 다른 자리에 있으면 볼
  - 같은 수가 전혀 없으면 낫싱
- 결과 문구 출력
  - 볼만 있는 경우 : `${n}볼`
  - 스트라이크만 있는 경우 : `${n}스트라이크`
  - 볼, 스트라이크 둘 다 있는 경우 : `${n}볼 ${n}스트라이크`
  - 하나도 없는 경우 : `낫싱`

### 3. 게임 종료

- 게임 종료 문구 출력 : `3개의 숫자를 모두 맞히셨습니다! 게임 종료`
  - 3개의 숫자를 모두 맞힐 경우
- 게임 재시작 문구 출력 : `게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.`
- 게임 재시작 여부 입력값 받기
  - 게임 재시작 : 1
  - 게임 종료 : 2
  - 예외 처리

### 예외 처리

- throw문을 사용해 예외를 발생
- 에러 문구 출력 : `[ERROR] ${에러 문구}`
- 애플리케이션은 종료
