## 1주차 숫자야구

## 요구사항
    
1. 게임 시작
- [x] 게임 시작을 사용자에게 알리기
    
2. 컴퓨터의 랜덤 수 생성
- [x] 컴퓨터의 수를 렌덤하게 생성

3. 사용자 입력
- [x] 사용자에게 입력 받기
- [x] 입력받은 수가 3개의 수가 아닌 다른 경우 예외처리
        (입력이 3개의 숫자가 아니거나 아무것도 안들어오는 경우)

4. 스트라이크 볼 확인
- [x] 컴퓨터의 수와 유저의 입력을 비교
- [x] 스트라이크 볼 낫싱을 반환


5. 결과
- [x] 입력을 처리 후 상태를 업데이트하여 출력
- [x] 정답시 재시작
- [x] 재시작 과정에서 1과 2 외의 수가 들어오면 예외처리
