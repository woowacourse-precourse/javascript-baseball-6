# 📄 기능 목록

- 입력 기능
  - [x] 사용자로부터 숫자를 입력받는 기능
  - [x] 사용자로부터 재시작 여부를 입력받는 기능
- 출력 기능
  - [x] 게임시작 문구를 출력하는 기능
  - [x] 사용자로부터 입력한 숫자의 결과 문구를 출력하는 기능
  - [x] 3개의 숫자 모두 맞을 경우 결과 문구를 출력하는 기능
- 추가 기능
  - [x] 상대방(컴퓨터) 숫자를 랜덤으로 생성하는 기능
  - [x] 상대방(컴퓨터) 숫자를 초기화하는 기능

# 🎯 예외 상황

- 숫자 입력

  - [x] 숫자가 3글자 이상인 경우
  - [x] 숫자가 아닌 다른 문자일 경우
  - [x] 중복되는 숫자가 입력있을 경우

- 재시작 여부 입력
  - [x] 1, 2가 아닌 다른 입력을 받을 경우
  - [x] 입력한 숫자가 1글자가 아닌 경우
