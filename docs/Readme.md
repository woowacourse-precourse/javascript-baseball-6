## 기능 구현 목록
1. 시작시 컴퓨터 숫자 생성
2. 숫자 3개를 입력해주세요 문구 생성
   - 숫자가 아닌 문자시 에러문구 출력
   - 3개의 숫자가 아니거나 0을 포함한 숫자 입력시 에러 출력 
3. 플레이어와 컴퓨터 숫자 비교
4. 숫자 비교후 스트라이크 및 볼 확인 없으면 낫싱
5. 위 2~4번을 반복
6. 게임이 끝나면 1과 2를 선택하여 재시작 및 종료를 결정
